# 制作bilibili弹幕词云视频

<img src="C:\Users\18295\Desktop\思维导图.jpg" alt="思维导图" style="zoom:50%;" />

## 1.you-get视频下载

利用 you-get 下载 B 站视频到本地  视频链接：https://www.bilibili.com/video/BV1rD4y1Q7jc?from=search&seid=10634574434789745619

## 2.爬取视频弹幕

利用you-get下载的弹幕XML文件 进行弹幕提取

## 3.从视频中提取图片

经过测试，这个视频完整分离出图片来有 3347 张，本文截取 800 张图片来做词云。

## 4. 利用百度AI进行人像分割

在百度云里创建一个人像分割的应用，对视频提取的图片进行人像分割

## 5. 小姐姐跳舞词云生成

利用词云库 对提取的人像进行填充

## 6. 合成跳舞视频

将填充完的图像进行视频拼接

## 7. 视频插入音频

在视频中插入原先下载的音频
